Mithila Bazaar is an e-commerce platform dedicated to promoting and selling authentic products from the Mithila region. The platform focuses on delivering high-quality, locally sourced items such as fox nuts (makhana), traditional Mithila paintings, homemade pickles (achaar), thekua, and other cultural specialties.

Our mission is to support local artisans and farmers while bringing the rich heritage of Mithila to a wider audience.

Features
 Online Shopping Platform – Browse and purchase traditional products
 Authentic Makhana (Fox Nut) – Directly sourced from Mithila region
 Handmade Mithila Paintings – Showcasing local art and culture
 Traditional Food Items – Achaar, Thekua, and more
 Order Management System – Smooth and secure order processing
 Responsive Design – Accessible across devices
 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Python
Database: MongoDB / MySQL
Tools: Git, GitHub

How It Works
Users browse products by category
Add items to cart
Place order securely
